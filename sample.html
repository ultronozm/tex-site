This file is intentionally small. It gives a fresh copy of the repository a publishable note immediately, and it exercises the same build path used by downstream sites.

Theorem 1. For every integer \(n\), the number \(n^2 - n\) is even.

Proof. We have \(n^2 - n = n(n-1)\), the product of two consecutive integers. One of those integers is even. ◻
